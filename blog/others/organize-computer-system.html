电脑系统相关整理 #

查看 IP 地址 #

  • window中在 cmd 中输入
sh
$ ipconfig
1
  • mac 中在 cmd 中输入
sh
$ ifconfig
1

截图快捷键 #

  • Print Screen截取全屏并保存在剪切板，在编辑器、文件夹、聊天窗口粘贴使用；（适合应用在截取右键菜单、快捷键导致鼠标菜单失效情况）
  • Win + Print Scree截取全屏并自动保存到 "此电脑>视频>捕获" 目录下；
  • Win + Shift + S调取系统截图工具，在编辑器、文件夹、聊天窗口粘贴使用；

本地 IIS 服务开启与启动 #

  • https://www.cnblogs.com/teyigou/p/8125379.html
  • 打开IIS管理器， win + R 输入 inetmgr
  • 本地访问地址：http://localhost:8099/、http://172.16.9.209:8099/
  • iis配置，其他电脑无法访问

windows 配置本地 hosts #

  • Win+R打开运行框，然后键入hosts文件路径：C:\WINDOWS\system32\drivers\etc
  • windows配置本地hosts

Mac 特殊符号 #

参考资料：Mac 下键盘特殊符号的那些事

说明

option 的意思是选择，所以在 Mac 给 option 多一种选择，当你在 Mac 下按下 option 时，总会有多一个选项，而这就是 option 的相关性。

  • option-按键 或者 shift-option-按键 来输入。
  • 使用“字符检视器”。可参考：在 Mac 上使用表情和符号
    • 在 Mac 上的 App 中，选取“编辑”>“表情与符号”,快捷键Control +  ⌘  + 空格；
    • 从输入法菜单中打开“表情与符号”（如果在“键盘”偏好设置中设定了该选项）。

输入法高效使用技巧 #

  1. 敲击 tab 键+部首，快速找到生僻字；
  2. 活用键盘符号；
    • xi'an (中文单引号)
    • Ctrl + 首字母
  3. 启用 u 模式，快速找到生僻字；
    • 中文输入： ushiwei 、uniuniuniu
  4. 启用 v 模式，秒变计算器;
  5. 启用简拼；
